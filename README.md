## Objective
To design and implement a star schema for analytical reporting using a retail sales dataset.

## Dataset
Global Superstore dataset (Kaggle)

## Tools Used
- SQLite
- Python (Pandas)
- Google Colab

## Schema Design
- Fact Table: fact_sales
- Dimension Tables: dim_customer, dim_product, dim_date

## Steps Performed
1. Loaded CSV dataset into SQLite
2. Created raw sales table
3. Designed dimension and fact tables
4. Inserted data using surrogate keys
5. Ran analytical queries using star schema joins
6. Exported analysis results

## Output
- analysis_outputs.csv

## Outcome
Implemented a complete star schema suitable for BI reporting and analytics.
